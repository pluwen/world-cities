# World Cities List

Multilingual list of countries, states & cities in XML format.

世界所有城市，国内所有省、市、区、县信息，不定期更新。

## Language support

| Language            | Code | File Name  |
| ------------------- | ---- | ---------- |
| Chinese Simplified  | 2052 | zh-cn.xml  |
| Chinese Traditional | 1028 | zh-tw.xml  |
| English             | 1033 | en.xml     |
| German              | 1031 | de.xml     |
| Spanish             | 3082 | es.xml     |
| French              | 1036 | fr.xml     |
| Japan               | 1041 | jp.xml     |
| Korean              | 1042 | ko.xml     |
